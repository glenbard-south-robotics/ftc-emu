# ftc-emu
An emulator for FTC robot OpModes

## Modules

| Path                   | Description                          |
| ---------------------- | ------------------------------------ |
| `ftc-emu`              | The main binary which wraps TeamCode |
| `ftc-emu-www`          | The local visualizer website         |
| `library/ftc-emu-core` | The WASM package for www             |
